# Quick Start

1. Open Unity
2. Go to **Tools → Clean Project → Open**
3. Click **Scan (Preview Only)**

Clean Project will analyze your project and list unused asset candidates.

---

## Moving Assets

- Select the assets you want to organize
- Click **Move Selected**
- Assets are moved to `Assets/_Cleanup`

No assets are deleted.

---

## Restoring Assets

- Open **Tools → Clean Project → Restore**
- Select assets
- Click **Restore Selected**

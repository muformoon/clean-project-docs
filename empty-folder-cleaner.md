# Empty Folder Cleaner

The Empty Folder Cleaner helps remove empty folders from your project safely.

---

## Features

- Detects folders that contain no assets
- Handles nested empty folders
- Optional detection of hidden folders

---

## Usage

1. Open **Tools → Clean Project → Empty Folder Cleaner**
2. Click **Scan**
3. Select folders
4. Click **Delete Selected**

Only empty folders are removed.

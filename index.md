# Clean Project

Clean Project is a safe and conservative Unity Editor tool designed to help you identify and organize unused assets in your project.

It does not delete assets.  
All unused assets are moved to a dedicated cleanup folder and can be restored at any time.

---

## Features

- Scan project for unused assets
- Preview results before making changes
- Move unused assets to a cleanup folder
- Restore assets to their original locations
- Clean empty folders safely
- Conservative handling of Resources folder assets
